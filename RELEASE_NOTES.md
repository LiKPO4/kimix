# Kimix v2.8.0 Release Notes

## 重大更新：独家功能「画板」上线

Kimix v2.8.0 是一个中等版本迭代，重点提升图片表达、需求澄清与 agent 沟通体验。现在你可以在对话输入区直接创建或编辑画板，把想法画出来再交给 agent，沟通更清晰、更流畅。

## 核心亮点

### 独家功能：简易画板
- 在输入区加号菜单中新增「画板」入口。
- 支持 `1:1`、`4:3`、`3:4`、`16:9`、`9:16` 五种画板比例。
- 已上传图片可直接打开画板继续编辑；基于图片编辑时使用原图尺寸作为画布。
- 保存后会自动作为 PNG 图片加入上传图片列表，继续发送给 agent。

### 绘图工具增强
- 支持画笔、橡皮、方形、圆角矩形、圆形、线条、裁剪、油漆桶。
- 支持 10 种颜色，并可切换前景色 / 背景色。
- 支持小、中、大、特大四档线宽。
- 支持撤销、重做与保存快捷键。

### 与 agent 沟通更清晰流畅
- 优化需求澄清中断后的会话状态处理。
- 修复首次澄清未完成后停止，再发送消息可能丢失、组合或重复显示的问题。
- 排队消息增加去重保护，减少重复发送风险。
- 图片 + 文字的沟通链路更适合描述 UI、草图和视觉需求。

## 快捷键

| 快捷键 | 功能 |
| --- | --- |
| `Ctrl+Z` | 撤销 |
| `Ctrl+Y` / `Ctrl+Shift+Z` | 重做 |
| `Ctrl+S` | 保存画板 |
| `B` | 画笔 |
| `E` | 橡皮 |
| `R` | 方形 |
| `Q` | 圆角矩形 |
| `O` | 圆形 |
| `L` | 线条 |
| `C` | 裁剪 |
| `F` | 油漆桶 |
| `1/2/3/4` | 小/中/大/特大线宽 |
