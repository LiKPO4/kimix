# Kimix v2.8.1 Release Notes

## 热修更新

本版本修复 v2.8.0 发布后的升级入口问题，并优化升级入口位置。

## 修复内容

- 修复点击「升级」时报错 `ensureDir is not defined` 的问题。
- 升级包现在会正确下载到用户下载目录下的 `Kimix Updates` 文件夹。
- 将「升级」入口从对话窗口工具栏移动到顶部菜单栏「帮助」右侧，更符合桌面菜单布局。

## 仍包含 v2.8.0 重大更新

- 独家功能「画板」上线。
- 支持从加号菜单新建画板，或从已上传图片打开画板。
- 支持画笔、橡皮、形状、线条、裁剪、油漆桶、颜色/背景色、线宽、撤销/重做和保存快捷键。
- 优化需求澄清中断后的会话状态处理，让与 agent 沟通更清晰流畅。

## 验证

- 已通过 `pnpm build`。
- 发布产物由 GitHub Actions 自动构建和发布，请以 CI 产物为准。
