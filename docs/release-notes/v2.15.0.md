# Kimix v2.15.0

## 修复

### 核心安全与数据完整性

- 文件撤销增加确认、作用域校验和回收站处理，避免误删用户后续修改或操作到无关项目。
- 定时关机状态持久化到主进程并支持恢复与退出提示，防止窗口关闭后倒计时和取消入口丢失。
- 绘画会话与待发送队列从 `localStorage` 迁出到 IndexedDB，图片单独存储并增加失败提示，避免配额耗尽导致数据静默丢失。
- IndexedDB 串行化保存与图片回收；读取失败时回退到 `localStorage`；复用 IndexedDB 连接；写入失败后保留队列下次重试。

### Hooks

- 修复项目根目录下项目级 Hook 不执行的问题。
- 禁止项目级 Hook 同步为全局 Kimi Hook，避免作用域承诺与实际行为不一致。
- 检测并拒绝灾难性回溯正则，覆盖 `((a+)+)` 等嵌套量词模式。
- Hook 规则生成按平台输出命令，并正确转义 Unix 单引号。
- 成功执行的 Hook 退出码记为 `0`。
- 非 `UserPromptSubmit` 事件禁止选择项目作用域。

### 搜索与文件

- `@文件` 搜索改为异步、可取消并加 600ms 时间预算，避免同步递归扫描阻塞主进程。
- 搜索结果恢复命令按平台生成，不再固定为 PowerShell。
- 跨平台路径大小写比较按平台处理，修复 Linux 下项目串号问题。

### 图片与附件

- 为图片粘贴增加数量、单张大小、总大小和并发读取限制。

### 更新与安装

- macOS 更新包按 CPU 架构筛选。
- 更新包下载后校验长度与 SHA256，缺失 SHA256 拒绝自动安装。
- 安装引导按平台展示命令。

### Markdown 与渲染

- 结构感知截断与展开状态保持。
- 截断预览时保护 LaTeX `$$` 数学块。
- 超长 Markdown 按需截断与展开；旧历史展开改为分批加载。

### 其他

- 允许 `data:` 字体和 `style-src-elem`，消除 CSP 控制台警告。
- ZIP 导入增加条目数、单条目与总体积限制。
- 同名技能按来源区分并启用；启用时去重并后覆盖前。
- 队列更多按钮增加排序菜单并支持键盘排序。
- 搜索历史会话失败时显示错误而非伪装空结果。
- 前后端统一文件预览扩展名白名单并即时校验。
- 设置与 Kimi 配置原子写入并保留备份；自动保存失败时通过 toast 提示用户。
- 长程任务状态文件原子写入并支持 bak 恢复。
- 语音快捷键与定时关机仅 Windows 展示。
- Escape 仅停止当前可见运行会话；修复 Escape 停止当前任务判断与 Composer 焦点下全局快捷键冲突。
- 为搜索与主要弹窗添加 dialog 语义和初始焦点，修复弹窗焦点陷阱。
- 搜索结果行改为 `div` 避免按钮嵌套；实现搜索 Ctrl+1-9 快捷键。

## 改进

- 补充 Markdown 截断、Hook 正则、哈希与状态持久化测试。
- 更新审计文档状态并回填修复提交 hash。

## 发布说明

本版本为中型更新，集中修复 2026-07-10 审计文档中确认的 P0/P1 缺陷，覆盖数据安全、Hook 作用域、持久化可靠性、跨平台一致性与搜索性能等核心领域。

由 GitHub Actions tag 流程构建和发布，不包含本地手动构建产物。
