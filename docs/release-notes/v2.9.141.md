# Kimix v2.9.141

- 修复长会话滚动时仍会在滚动帧内触发消息锚点 DOM 扫描的问题，改为滚动停顿后再捕获锚点。
- 优化聊天流 ResizeObserver：尺寸变化统一合并到 `requestAnimationFrame` 内处理，减少同步布局读写抖动。
- ResizeObserver 优先只观察消息内容容器，窗口尺寸变化走独立 resize 兜底，降低滚动容器本身反复触发观察回调的概率。
- 用户消息气泡只订阅显示所需的最小运行态，重发时再读取默认思考、Plan 和权限配置；引导气泡增加 memo，减少长会话运行中刷新时的无效重渲染。
- MessageBubble 增加内容级 memo 比较，避免 `buildRenderItems` 生成的新数组引用让历史消息无意义重渲染。
