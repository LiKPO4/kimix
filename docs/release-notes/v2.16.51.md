# Kimix v2.16.51

本版本跟进 Kimi Code 0.24–0.27（Server v2 引擎与本地回退运行时），集中修复运行状态误判、消息头消失、对话历史错乱与视口跳动四类长期问题，并带来对话导航轨道等交互改进。

## 修复

### 运行状态与消息头

- 以官方 `busy` 信号判定 Server 运行态（整个任务期间、含步骤间隙均保持有效）；未知或畸形状态不再被误判为"已完成"，消除运行中闪现"输出完成"。
- 运行中的最新一轮不再被渲染缓存提前收口，消息头不再"闪断后重现"，也不会在任务未结束时提前显示"输出完成"。
- 消息发出后到"思考中"之间的空窗期保留消息头，不再短暂消失。

### 对话历史

- 修复快照历史回放把用户消息整体复制多遍的问题（个别会话被重复消息淹没、回复被挤出可视区）；启动时自动清理已产生的重复。
- 回放合并后按时间恢复正确顺序，旧内容不再错位拼进最新一轮（"一轮回复里夹杂多轮内容"）。
- 历史回放中的助手回复不再被错误合并进当前轮，跨轮内容污染得到根治。

### 视口与滚动

- 滚动锚定改为按"跟随/查看历史"模式区分：跟随底部时恢复浏览器原生锚定，启动进入会话直接落底、流式期间不再反复上跳、点"置底"后稳定贴底。
- 恢复视口尾部补偿，过程折叠与展开时视口不再跳动或残留空白。

### 兼容与稳定

- 跟进 Kimi Code 0.24–0.27：适配 Server v2 鉴权与协议，本地回退运行时同步至 0.27（保留 MCP 启动超时等本地补丁），Windows 路径大小写与斜杠变体归一为同一工作区。
- Server 单例锁在 PID 复用时自愈，不再错误停留在回退模式；开发重启脚本不再误杀正式安装版。
- 长历史会话的用户消息去重性能优化，避免卡顿。

## 新增

- 对话导航轨道：右侧可视化定位到任意消息，悬停预览内容。
- 空项目点击时自动创建对话；欢迎屏底部显示默认模型并允许预选择。

## 验证重点

- 长工具链对话：标题持续显示运行态，不提前、不闪现"输出完成"，消息头不消失。
- 历史长的会话：重开后回复完整、顺序正确，无重复消息、无跨轮混杂内容。
- 启动进入会话直接落底；流式期间视口稳定，点"置底"后保持贴底。
