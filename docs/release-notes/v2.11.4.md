# Kimix v2.11.4

## Kimi Server slash 路由

- Server 会话下优先把 `/goal`、`/swarm`、`/compact`、`/undo` 等官方 slash 命令原样交给 Kimi 官方链路处理。
- 仅保留 Kimix 专属命令本地处理：`/theme`、`/custom-theme`、`/import-from-cc-codex`。
- 官方发送失败时再尝试旧的 Kimix 本地兜底逻辑，减少 Server 能力被前端提前截断的问题。
- 新增 `pnpm probe:kimi-server-slash`，用于实测官方 Server 是否接收并完成低风险 slash 命令；高风险命令需显式开启 mutating 探针。
