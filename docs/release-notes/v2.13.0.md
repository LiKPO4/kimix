# Kimix v2.13.0

## 新功能

- 跟进 Kimi Code 0.22.0，将兼容回退 SDK 更新到官方 Node SDK 0.12.0。
- `Ctrl/Cmd+K` 现在打开全局会话搜索，可按标题、项目路径和最近提示词快速切换会话。
- 模型目录保留官方提供的思考强度档位和默认 effort 元数据。

## 修复与优化

- 第三方 OpenAI 兼容模型缺少输出上限时，将修正值持久化到官方 `models.<alias>.overrides`，避免供应商刷新后丢失。
- 兼容 SDK 0.12 的 `thinkingEffort` 状态字段，并继续向界面提供稳定的思考强度状态。
- Server 图片提示继续使用官方 0.22 的提交前自动压缩链路；SDK 回退路径同步获得相同能力。
