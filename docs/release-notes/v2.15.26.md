# Kimix v2.15.26

## 多 Agent 停止与引导控制

- 房间 Stop 和 Steer 现在使用目标 Agent 的 runtime，不再默认操作 primary Agent。
- 多个 Agent 同时运行时会先显示明确的 Agent 列表；用户选择后才执行，Stop 额外支持逐 Agent 汇总的“停止全部”。
- Steer 消息写入目标 Agent 当前 turn 的事件分区，其他 Agent 的历史、运行状态和 delivery 不受影响。
- Stop 只结算目标 Agent 的 Assistant、工具、审批、提问、Steer 和 delivery，并保留其他 Agent 继续运行或排队。
- 活动表状态优先于较旧的 delivery 状态；只有活动记录暂时缺失时才使用持久化 delivery 恢复控制入口。
