# Kimix v2.10.0

## Kimi Code 0.17.1 Server

- 默认启用官方 Kimi Server 路由，失败时自动回退 SDK，并在界面展示实际发送链路和当前阶段。
- 接入 Server 会话恢复、prompt、steer、取消、审批、提问、压缩、撤销、BTW、归档、用量和上下文状态。
- 补齐 Skill、MCP、会话树、工具目录、连接诊断、后台任务输出、消息与 prompt 队列诊断。
- 增强 WebSocket 断线重连、序号重放、snapshot 正文恢复和历史消息去重。

## 性能与稳定性

- 优化长会话滚动和消息渲染，降低滚动热路径 DOM 扫描与流式事件重渲染开销。
- 修复向上阅读时 ResizeObserver 锚点补偿造成的进度倒退。
- 优化首次发送的 runtime 预热与握手耗时，老会话失效 runtime 会自动恢复。
- 稳定运行计时和终态收口；漏收完成事件时会通过底层状态对账恢复发送按钮。
