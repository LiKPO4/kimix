# Kimix v2.15.16

## 重点更新

- Kimi Code 运行时对齐官方 `0.23.5` / Node SDK `0.13.2`；新会话固定采用自动 Server 优先，能力不足或请求失败时仍会自动回退 SDK。设置页不再暴露容易造成误解的 Server 路由开关，实验性工具按需加载继续保持默认关闭。
- 长会话浏览更稳定：历史过程默认折叠并按需展开，工具输出省略区可继续展开；运行中的手动滚动、底部跟随、模型切换后的新轮次以及本地图片在官方历史刷新后的预览均已修复。
- 桌面通知新增待审批工具操作提醒，并遵守现有通知模式和隐私开关；点击完成、澄清或审批通知会恢复窗口并回到对应项目会话。
- “套餐用量”浮层支持官方 Extra Usage（BOOSTER 钱包）只读展示，包含本月已用、月度上限/不限额与可用余额；五小时和本周的绿色时间条统一使用“已过 N%”文案。

## 体验改进

- 文件变更卡片的展开/收起 hover、路径展示和工具省略提示更一致。
- 项目侧栏操作仅在悬停或键盘焦点时显示；置顶与取消置顶整合为同一书钉操作，操作组更紧凑且保持右侧对齐。
- 修复会话持久化旧快照覆盖、Intel Mac 旧更新资产识别与运行耗时的无效高频刷新。

## 验证

- `pnpm test:run`：60 个测试文件、431 项测试通过。
- `pnpm build`、`pnpm knowledge:validate` 与 `git diff --check` 通过。
- 安装包由 GitHub Actions 在标签推送后自动构建和发布，未包含本地手动构建产物。
