# Kimix v2.14.30

## 改进

- `+` 菜单的“需求澄清”下方新增 Swarm 模式入口。
- Swarm 作为会话级能力开启：开启后锁定当前会话，后续消息继续在同一个官方会话 ID 中运行。
- 空闲 Server 会话开启 Swarm 时，会在同一个 session id 下切到 SDK 兼容路由，不 fork、不创建同名对话。

## 边界

- 当前轮运行中不能开启 Swarm，需要等本轮结束后再操作。
- Swarm 开启后本会话不允许关闭，避免 UI 状态与官方运行时状态漂移。
- 官方 Server 仍未公开 Swarm REST API；本版本使用同 ID SDK 兼容链路承接该能力。
