# Kimix v2.14.79

## 功能

- 文本选中右键菜单新增"翻译"入口：复制 `将以下内容翻译成中文：...` 提示词并打开豆包网页版。
- 消息队列优先使用官方队列能力，支持展示、删除、重排，并保留排队消息附件编辑状态。
- 接入官方 `select_tools` 实验开关和模型能力展示。
- 支持官方归档会话恢复，兼容 Web 端恢复归档会话后的本地侧栏同步。
- Swarm 展示统一收敛到消息流 inline 过程卡，Composer 底部仅保留精确状态按钮。

## 修复

- 修正中文标点后的链接识别，避免把后续中文半句错误标蓝。
- 展开项目时同步补齐会话目录，启动恢复时优先回到上次真实会话，避免跳到其他项目会话。
- 中断发送、失败终止和运行态收尾更稳定，避免消息长期显示"处理中"或底部运行态不释放。
- 套餐用量获取改走官方 SDK，并恢复用量刷新时间显示。
- 运行中上下文压缩不再误报超时，压缩完成后展示官方返回的可读 summary。
- 图片附件发送时保留绝对路径和本地预览信息，降低刚发送即丢失图片路径的概率。
- 图片预览遮罩层级提升，修复部分区域点击后无法退出预览。
- Kimi 模型默认保留官方 preserved thinking；DeepSeek / 第三方模型按能力降级，且保留 thinking signature 元数据。
- Plan 审批、Bash/Edit 工具过程卡和官方提问答案格式保留更多官方字段，减少 Kimix 本地兼容层造成的信息丢失。
- `@` 文件补全覆盖额外工作目录和深层文件。

## 改进

- 收紧侧栏滚动区左右留白和滚动条右侧间距，让项目/会话选中态阴影延伸更自然。
- 降低思考详情字号，并让主界面字号设置实际生效。
- Kimi Web 过程视觉巡检完成：Swarm inline、问题卡、文件补全、队列卡在 Kimix / Kimi Web 两种过程模式下避免重复展示。
