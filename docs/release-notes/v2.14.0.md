# Kimix v2.14.0

## 新增与跟进

- 跟进 Kimi Code 0.22 SDK，支持 `thinkingEffort` 字段，自动映射到稳定 `thinkingLevel` 状态。
- 为 OpenAI 兼容模型自动补充保守的 `maxOutputSize` 限制。
- 全局搜索面板合并原有独立的会话搜索对话框，支持键盘上下选择、回车打开、跨项目搜索历史会话。
- 文本选中右键菜单：复制、全选、打开链接，并识别本地路径点击在文件管理器中定位。
- 用量面板：添加剩余时间绿条、时间/颜色/高度调整、打开面板时静默刷新。
- markdown 行内代码和段落添加长内容换行保护，防止横向溢出。

## 修复

- 修复中文默认标题“新会话”被误当作自定义标题，导致 Skill/Fork 会话初始显示占位名的问题。
- 修复首帧侧栏出现大量未确认空占位、项目切换后短暂闪烁的问题。
- 修复会话目录行标题优先使用首条提示覆盖官方自定义标题的问题。
- 修复侧栏时间显示运行时元信息而非最后对话时间的问题。
- 修复会话标题在历史加载前不同步、归档后复活、SDK 会话归档失败等问题。
- 修复消息流撑不满页面时置顶而非置底、打开会话停顶部、上滚闪跳、内容收缩时底部锚定偏移等系列滚动问题。
- 修复全局搜索异步加载历史完成后可能覆盖用户已切换的会话的问题。
- 修复页面刷新后不恢复到原会话的问题。

## 发布说明

本版本涵盖 7 月 2 日~3 日的密集修复，主要包括会话系统稳定性重构（缓存/目录/标题/归档/占位/activity 时间）、滚动系统重新稳定、全局搜索合并、SDK 升级到 0.22 及模型配置适配。

由 GitHub Actions tag 流程构建和发布，不包含本地手动构建产物。
