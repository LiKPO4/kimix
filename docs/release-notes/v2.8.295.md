# Kimix v2.8.295 Release Notes

## 小型更新：修复安装登录引导与对话状态显示

本次更新修复 0.9.0 适配后的几处高频体验问题，重点改善安装 Kimi Code 后的登录路径、运行中状态显示和思考内容排版。

### 安装后登录引导
- 旧版 Kimix 更新后安装新版 Kimi Code 成功时，自动进入设置页并聚焦到 `Kimi 登录` 区域。
- onboarding 和更新弹窗中的登录指引改为使用 Kimix 设置页登录，不再要求用户手动去终端运行 `/login`。
- 进入登录区时会刷新登录状态，避免安装后仍看到旧状态。

### 运行中状态
- 修复 assistant 显示“执行中”时，本轮耗时不显示的问题。
- 活跃轮次判断同时兼容 Kimix 本地 session id 和官方 runtime session id，减少状态行、空态和停止按钮判断不一致。

### 思考内容展示
- 思考展开内容改用 Markdown 渲染，编号列表、代码和普通段落更易读。
- 对 `1. ...2. ...3. ...` 这类编号粘连做显示层换行修正，不改原始会话数据和导出内容。
- 明确外层代码块内嵌三反引号属于 Markdown 标准闭合行为，Kimix 不做非标准嵌套兜底。

## 验证

- `pnpm build` 通过。
- `pnpm test:run` 通过：6 个测试文件、83 个测试全部通过。
- 发布产物由 GitHub Actions 自动构建和发布，请以 CI 产物为准。
