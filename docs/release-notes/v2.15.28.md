# Kimix v2.15.28

## 房间外部会话操作绑定当前 Agent

- 底部模型选择器现在显示并切换当前唯一接收 Agent 的模型；零选或多选时不会再默认修改 primary。
- 右侧会话栏的官方 Goal、BTW、Plan 状态使用当前 Agent 的独立 runtime 和持久状态，并明确显示 Agent 名称。
- 顶部会话菜单中的官方重命名、派生、Kimi Web 和会话可视化均作用于当前唯一 Agent；派生历史和模型来自该 Agent，不混入房间其他成员。
- 运行门禁改为读取目标 Agent 的独立活动态，次要 Agent 长时间运行时也不能误切模型、派生或发起 BTW；其他空闲 Agent 仍可独立操作。
