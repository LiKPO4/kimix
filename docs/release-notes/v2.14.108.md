# Kimix v2.14.108

## 修复

- 图片预览键盘监听改为 overlay 容器内捕获，避免全局方向键被劫持导致无法在输入框/搜索中使用方向键。
- 模型切换过程中底部优先显示目标模型（`switchedToModel`），避免 API 返回前残留旧模型名称误导用户。
- 无焦点模式（通知设置）改为仅在窗口真正失去焦点时才发送系统通知；窗口被其他应用遮挡时不再被误判为「用户可见」，该提醒时不再静默。
- 模型切换按钮的禁用范围从「任一会话有 open timeline work」收窄为「仅当前实际运行中的会话」，避免切换到其他会话时模型选择仍被锁死。
- 套餐用量刷新时间解析：同时支持 `reset_in` 秒数、`resetHint` 可读文案（如 "resets in 3h"、"resets at 8:00"）和 API 实际窗口时长（`duration` + `timeUnit`），进度条绿色时间格不再依赖 label 推测而是真实窗口。
- 侧栏会话列表转圈动画同时检查 `hasOpenTimelineWork`（无超时），避免多会话并发时长轮次超过 2 分钟后丢失 loading 指示器。
- 上下文压缩完成事件改为紧接在 begin 事件后插入，不再被排到当前轮次 agent 输出下方，视觉上压缩过程的起止连贯可读。

## 改进

- 上下文压缩完成摘要的标题居中，保持与正文层级区分。
- 图片预览导航匹配逻辑提取为 `findPreviewImageIndex` 纯函数并补 5 个边界用例（id/dataUrl/空列表/无匹配）。
- 清理 `parseDurationToSeconds` 正则中无用空格字符。
