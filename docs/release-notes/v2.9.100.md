# Kimix v2.9.100

## 修复

- 引导消息不再把 Kimix 本地合成的 fallback 记录显示为“成功”，只有官方 `turn.steer` 记录出现后才显示为已记录。
- 引导气泡文案改为“等待官方确认 / 官方已记录引导”，避免误导用户以为 agent 已经实际采纳了引导内容。
- SDK 新链路的工具结果重新解析结构化 `diff` block，差异面板可以再次显示文件变更。
- 结构化 diff 会同步生成本轮文件变更摘要，恢复对话末尾的文件变更卡片。
- 当官方 `Write/Edit` 工具没有返回结构化 diff 时，也会基于工具调用路径生成文件变更摘要卡片。

## 验证

- `pnpm test:run`
- `pnpm build`
