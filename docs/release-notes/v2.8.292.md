# Kimix v2.8.292 Release Notes

## 中型更新：适配 Kimi Code 0.9.0

本次更新围绕官方 Kimi Code 0.9.0 做兼容适配，升级 vendored SDK，并接入 0.9.0 新增的 BTW 侧通道能力。

### Kimi Code 0.9.0 / SDK 0.7.0
- vendored SDK 更新到官方 Kimi Code 0.9.0 对应的 node-sdk 0.7.0。
- 兼容新版 `createKimiHarness` 创建方式，同时保留旧构造方式兜底。
- 更新 SDK 探针和 provenance 文档，记录 0.9.0 tag、构建和已知 Windows dts 问题。

### BTW 侧问
- 接入官方 `Session.startBtw()`，右侧栏新增 BTW 侧问入口。
- BTW 侧问不打断主轮次，侧通道事件按 agent id 隔离，不混入主对话。
- 侧问记录按“轮”展示，最新在上，支持 Markdown 渲染、折叠和本地持久化。
- 侧栏输入快捷键与主输入区一致：Enter / Ctrl+Enter 发送，Shift+Enter 换行。

### 对话和长程任务稳定性
- 修复同会话队列续发时可能并发发出多条消息的问题。
- 修复后台 agent 完成、停止和恢复提示的状态展示。
- 补齐长程任务侧栏中后台任务输出、停止状态和待审查入口。
- 修复被打断输出仍显示“输出完成”的状态文案。

### 设置和模型配置
- 优化设置页为更紧凑的横向行式布局，减少左右列高度失衡。
- OpenAI-compatible Provider 支持从官方 catalog 填入，并对 DeepSeek 等模型 Context 上限做收敛。
- 模型卡片文案从“默认”调整为“使用 / 使用中”，降低用户理解成本。
- 设置页连接状态、登录状态、模型配置增加缓存，避免进入设置时闪烁。

### 体验修复
- 画板打开已有图片时，原图作为背景层；油漆桶只作用于用户绘制层。
- 更新弹窗中“浏览器下载”移动到对应按钮下方，减少按钮挤压。
- 顶部小窗和文件小窗支持点击外部关闭。
- 对话复制旁新增“全部”按钮，可复制回复和思考内容。
