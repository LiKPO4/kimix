# Kimix v2.9.44

## 修复

- 修复官方 SDK 仍有 active turn 时，Kimix 发送失败分支会提前清空运行态、导致输入框误解锁的问题；现在会恢复运行态并提示等待当前轮结束。
- 修复上下文压缩/窗口恢复后误发新消息留下本地气泡和空执行头的问题；官方拒收的新 turn 会回滚本地占位，队列消息会放回待发队列。
- `sendKimiCodePromptWithRetry` 不再在多次 active-turn 拒绝后自动取消官方当前轮次，避免误停真实仍在执行的任务。
- 收窄 `scripts/restart-kimix-dev.ps1` 的进程匹配范围，只处理命令行包含当前仓库路径的 dev 进程，避免误碰安装版或其他 Kimix 相关进程。
- 版本号三处同步到 v2.9.44：`package.json`、侧栏、设置页。
