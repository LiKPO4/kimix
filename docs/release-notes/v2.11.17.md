# Kimix v2.11.17

- 优化 Kimi Code Server 优先链路：可原生走官方 server 的 slash 能力优先使用官方链路，无法迁移的能力保留 Kimix 兜底。
- 清理旧版 Kimi IPC / SDK 对接残留，统一 renderer 会话 API 到 Kimi Code 命名空间，降低历史接口误触风险。
- 改善启动与恢复体验：减少启动白屏等待，启动脚本默认使用更适合开发的热更新模式，并补充诊断日志输出。
- 简化发送状态文案，隐藏自动重试和活跃轮次等内部状态，避免把底层链路细节暴露到消息头。
- 修复助手消息底部信息气泡缺失，以及“消息发送中 / 正在提交给 Kimi Code”状态被错误渲染进聊天流的问题。
- 部署并自动化维护 OKF 项目知识库，新增严格校验、每周审计和可复用部署 Skill。
