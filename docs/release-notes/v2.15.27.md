# Kimix v2.15.27

## Composer 会话操作绑定唯一 Agent

- 房间权限、Plan、Goal、Swarm、Skill、Plugin 和直接 Slash 命令现在只作用于接收者中唯一选中的 Agent；零选和多选时均明确拒绝，不再默认使用 primary 或焦点 Agent。
- Agent 级持久状态补齐 `permissionMode` 和 `planMode`，与现有模型、Swarm、Goal、BTW 状态一起进入独立 session view、备份和 primary 兼容镜像。
- `/compact`、`/reload`、`/status`、`/usage`、`/btw` 和 `/undo` 使用目标 Agent runtime；本地命令、结果和错误只写入目标 Agent 事件分区。
- 单 Agent `/undo` 已支持房间次要 Agent 的 canonical snapshot，只替换目标历史，不影响其他 Agent。
- 运行中的房间 Agent 禁止切换权限；空闲 Agent 可独立修改，Plan 和 Swarm 的按钮状态显示当前唯一目标的真实值。
- 兼容旧单 Agent 会话时保留当前权限模式；官方自定义主题 Skill 失败转兼容兜底时不再重复显示同一条命令。
