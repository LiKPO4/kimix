# Kimix v2.14.107

## 功能

- **Swarm 随时开关**：Swarm 模式不再锁定整个会话，支持运行中切换（下一轮生效）。关闭亦不需要重建会话。
- **Swarm 聚合状态卡**：Swarm 子代理聚合展示进度条、完成/失败计数和子代理详情，状态一目了然。
- **更新记录完整展示**：关于弹窗的更新记录改用 Markdown 完整渲染 Release body，不再截取一行摘要。
- **多图预览**：多图消息支持方向键切换上一张/下一张，Esc 可退出预览，切换有边界保护。
- **官方归档会话恢复**：从 Kimi Web 端归档的会话可以在 Kimix 中恢复并继续。
- **官方消息队列优先**：优先使用 Kimi Code 官方消息队列能力，支持展示、删除、重排，队列编辑默认恢复为普通输入。
- **select_tools 实验开关**：接入官方 `select_tools` 实验功能及模型能力展示。
- **文本翻译**：右键菜单新增"翻译"入口，一键复制提示词并打开豆包网页版翻译。

## 修复

### 会话与启动
- 启动恢复优先回到上次真实会话，不再跳到其他项目会话。
- 展开项目时同步补齐会话目录，避免已存在会话不显示。
- 修复 skill fork 会话启动时未正确加载内容的问题。
- 偏好开关（思考/Plan/权限）更改不再触发会话恢复重启，避免中断当前连接。

### 聊天滚动
- 修复长会话恢复后滚动错位和输入区不可交互问题。
- 聊天壳固定到 Electron 视口，避免容器的 CSS 布局竞态。
- 思考块折叠/收缩时不再触发 scrollToBottom 猛跳或向上 undershoot。
- 修复 scrollToBottom 诊断日志作用域导致的 ReferenceError。

### Swarm 与子代理
- 迟到子代理完成事件不再丢失，合并回已结束的子代理卡片。
- 子代理内部编号不再暴露到 UI（兜底名统一为"子代理"）。
- 迟到事件不复用兜底名覆盖原有具体名称（如 "coder"→"子代理" 退化）。
- 修复 Swarm 进度显示和下载更新时的降级处理。
- Swarm 展示统一收敛到消息流 inline，右侧栏状态按钮仅保留精确同步状态。

### 思考与内容
- 从官方历史恢复完整的思考摘要，缓存版本升到 v2，避免旧缓存里思考文本重复。
- 保留 Kimi 默认思考偏好，DeepSeek / 第三方模型按能力降级，保留 thinking 签名元数据。
- 对齐官方 Plan 审批、Bash/Edit 工具过程卡和提问答案格式，减少信息丢失。
- 显示上下文压缩完成后官方返回的可读摘要。
- 恢复正文默认字号，并让主界面字号设置实际生效。

### 图片与附件
- 从官方历史正确恢复图片数据，保留绝对路径和本地预览。
- 图片预览遮罩层级提升，修复部分区域点击后无法退出预览。
- `@` 文件补全覆盖额外工作目录和深层文件。
- 更新弹窗剥离 Release body 中的 markdown 图片，避免弹窗自动加载外部图片。

### 布局与 UI
- 顶部工具栏和按钮避让右侧栏，窄侧栏 Git 按钮自适应换行。
- 项目行点击仅控制展开/折叠，不再误触发其他操作。
- 套餐用量获取改走官方 SDK，净化错误信息并恢复刷新时间显示。

### 运行与中断
- 中断发送时彻底清理运行态，避免消息长期显示"处理中"。
- 运行中上下文压缩不再误报超时。
- 修正中文标点后的链接识别，避免后续中文字符被标蓝。

## 改进

- 更新弹窗剥离 Release body 图片，防止隐私跟踪和弹窗卡顿。
- 思考详情字号降低，与正文拉开层级。
- 思考按钮文案间距统一；翻译按钮简化为"翻译"。
- 单段过程与思考总结在 Kimix / Kimi Web 双模式下视觉对齐。
- 不可展开过程文本与正文字号和行距统一。
- 侧栏滚动区左右留白和滚动条右侧间距收紧，选中态阴影延伸更自然。
- 队列消息保留附件编辑状态（拖拽图片不丢失），本地重排对齐官方 Web 行为。
