# Kimix v2.8.364

## 修复与改进

### Git 工具箱

- 在右侧会话侧栏加入 Git 工具卡片，支持查看当前分支、工作区改动、远端、upstream、ahead/behind 状态。
- 增加 Git 详情面板，可按文件勾选本次提交范围，并提供提交、拉取、推送等常用操作。
- 修复 `git status --porcelain` 解析时丢失前导空格的问题，避免文件名首字母被误删。
- 区分“未配置远端”和“已配置远端但未设置 upstream”，有远端但无 upstream 时推送会尝试自动设置跟踪分支。
- 修复切换项目后 Git 详情读取旧目录、远端状态刷新不及时的问题。
- 优化 Git 详情打开体验，避免点击详情时短暂闪过默认文本。
- 统一右侧 Git 工具的项目语义：点击会话后，Git 状态跟随该会话对应的项目目录。
- 在右侧 Git 卡片标题行显示当前项目名，帮助确认 Git 操作目标；项目名过长时单行省略，避免挤压左侧标题。

### Kimi Code 与会话

- 修复 Kimi Code 正文回填丢失问题，兼容新的 `context.append_loop_event -> content.part` 事件形态。
- 启动后尝试从官方会话历史回填正文，降低历史消息只剩状态块的概率。
- 优化会话与项目同步：点击会话时同步切换到该会话对应项目，避免右侧工具读到不一致的项目状态。
- 精简权限模式提示，并同步到权限悬停说明。

### 长程任务

- 移除审查 agent 流程，长程任务改为执行 agent 连续推进，最终交给用户全盘审查。
- 普通对话不再显示长程任务卡片，已归档任务不再出现在长程任务列表里。
- 隐藏过大的长程任务执行提示内容，减少右侧侧栏视觉负担。

### 右侧工具箱与设置

- 右侧工具箱卡片支持拖拽排序并持久化。
- 设置页分区支持拖拽排序并持久化。
- 将工具类状态入口整理到右侧会话侧栏，减少左下角杂散入口。
- 修复右侧侧栏若干文案、对齐、空白和刷新闪烁问题。

### 卡死诊断

- 增加 renderer 卡顿 watchdog，由主进程记录疑似卡死现场。
- 设置页卡死诊断条目支持导出日志，便于后续精确定位卡死来源。
- 优化诊断内容展示，避免内部错误信息直接大段糊到界面上。
