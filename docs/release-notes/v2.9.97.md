# Kimix v2.9.97

## 会话迁移

- 停止 dev 版和安装版自动共享会话数据，改为显式“会话迁移”。
- 设置页新增“导出全部 / 合并导入”，支持拖入 `.zip/.json` 快照直接合并。
- 快照包含会话、归档状态、待发送队列、项目、隐藏交接记录和 active context。
- 导入时按会话 id、官方 id、runtime id 去重；当两边同一会话都各自新增内容时，导入内容会生成“导入副本”，避免硬塞进同一条时间线。

## 交互修复

- 展开“已折叠较早对话”时保持当前可视位置，不再跳到会话顶部。
- Agent 正在输出正文时发送新消息会进入待发送队列，避免两轮内容混杂。
- 侧栏多个会话同时运行时，会分别显示加载转圈，不再只有最新会话显示运行态。
- 设置页归档对话“展开剩余”数量改为按实际未展示条目计算，展开后显示“收起归档列表”。

## 导出与渲染

- 收敛聊天记录 Markdown 导出入口，顶部状态栏和侧栏 Markdown 图标都走同一条导出路径。
- 侧栏 Debug ZIP 导出改名为“Kimi 调试包”，避免和 Markdown 导出混淆。
- 修复流式输出把 Markdown 表格分隔线拆成多段时，表格无法被 `remark-gfm` 识别的问题。
