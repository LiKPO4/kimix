# Kimix v2.9.99

## 修复

- 补齐默认项目显示名兜底，空态页、长程任务入口和检查面板也会显示“Kimix 默认项目”。
- 会话快照合并导入时保留本机隐藏内部会话，避免导入动作误清正在运行的内部状态。
- 当前轮仍在输出时，输入 slash 或 skill 命令也会先进入待发送队列，避免绕过普通消息的排队保护。
- 会话快照导入/导出的文件读写改为异步路径，降低大快照操作时主进程卡顿风险。
- `useLiveSession` 改为复用会话 id 索引，减少同一轮状态更新里多个组件重复扫描会话数组。
- 设置页归档摘要不再通过 JSON 字符串往返计算，并用浅比较减少无关会话更新带来的重渲染。
- 侧栏按项目路径缓存可见会话分组，避免每个项目渲染时重复扫描全部会话。
- 本地持久化订阅在判断归档变化时改用 id Map，减少会话多时的重复查找。
- 已完成的大段 assistant Markdown 离视口较远时延迟渲染，接近视口后再执行 Markdown/代码高亮解析。

## 验证

- `pnpm test:run`
- `pnpm build`
