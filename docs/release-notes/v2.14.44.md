# Kimix v2.14.44

## 修复

- 修复 AI 仍在工作时刷新页面会把当前轮错误显示为已完成、随后发送触发 active turn 冲突的问题。
- 修复 Server snapshot 将仍在生成的 Assistant 正文错误标记为 `turn.ended` 的问题。
- 运行中正文事件长时间没有抵达界面时，会节流回读官方 snapshot，并在内容更完整时恢复显示。

## 验收建议

1. 在 AI 持续输出正文时按 Ctrl+R，界面应继续保持运行态，输入区不能启动冲突的新一轮。
2. 模拟正文数秒没有更新，确认后续官方 snapshot 能补回新增正文。
3. 等官方 turn 真正结束后，界面才进入完成态并允许发送下一条消息。
