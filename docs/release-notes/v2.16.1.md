# Kimix v2.16.1

本版本清除了 Electron 主进程、预加载层、渲染端和测试代码的 TypeScript 严格类型基线，并新增 `pnpm typecheck` 双边界门禁。

## 修复

- 修复 SDK 兼容链路 BTW 侧问缺少 Agent 上下文执行器的问题，并确保回退路径恢复原 Agent 上下文。
- 修复空白页快捷发送失败时异常清理可能再次引用越界变量的问题。
- 移除消息分组中的互斥判断和不可达 Assistant 分支，保持消息归属按既有时间线规则渲染。
- 修复“从不显示状态更新”设置无法通过 IPC 校验持久化的问题。
- 校准 Electron 对话框、通知窗口、后台任务、会话目录和事件载荷的空值及联合类型边界。

## 工程质量

- TypeScript 目标与项目实际使用的 ES2023 API 对齐。
- `pnpm typecheck` 串行检查 Electron main/preload 与 renderer/shared/tests；生产构建不再代替类型安全证据。
