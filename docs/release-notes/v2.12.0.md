# Kimix v2.12.0

## 新增与跟进

- 跟进 Kimi Code 0.19/0.20 的运行时与 Web UI 变化，包括新快照结构、后台任务边界、LaTeX 行间公式、官方 0.20.0 变更记录与相关兼容说明。
- 顶部工具栏新增 Kimi Web 浏览器入口，保留项目菜单中的“打开终端”，并避免自行打开未认证的会话深链。
- 支持刷新页面快捷入口，并在重新载入后尽量恢复到原会话。
- 增加前端整齐度与视觉节奏整理基线，统一部分侧栏、插件、弹窗和聊天区域的间距与对齐。

## 修复

- 修复手动发送消息后不主动滚到底部的问题。
- 修复消息正文下方元信息气泡偶发空白、system-reminder 被误显示为用户消息、长会话首次打开不置底等问题。
- 修复恢复会话、队列冲突、已存在 server session 接管、缺失 archive 返回等 Kimi Code 会话边界问题。
- 修复刷新页面后不回到之前会话的问题。
- 修复点击语音快捷键后输入框丢失焦点的问题。
- 增强 Kimi Web server 断连恢复，降低长时间打开后莫名断连的影响。
- 修复侧栏折叠按钮异常、技能标题裁切、工具栏刷新按钮语义不清等 UI 问题。
- 修复 Kimi Web 入口先打开正常页面、又额外打开需要 token 的标签页的问题。

## 发布说明

- 本版本由 GitHub Actions tag 流程构建和发布，不包含本地手动构建产物。
