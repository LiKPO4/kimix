# Kimix v2.16.2

本版本修复项目折叠状态与上次活动项目、会话在重启后的恢复冲突。

## 修复

- 项目展开与折叠状态按规范化项目路径持久化，项目 ID 变化不会丢失偏好。
- 用户明确折叠当前项目后，重启不再因为恢复活动项目而强制展开。
- 首次使用没有已保存偏好时，仍自动展开恢复的当前项目。
- 保存了项目但没有活动会话时，启动不再跳到其他项目的最近会话。
- 启动期间的临时最近项目不再覆盖退出前保存的活动上下文。
- 设置加载只负责读取设置和项目目录，活动项目由完成本地会话水合后的恢复流程统一决定。

## 验证

- 新增项目展开状态序列化、启动上下文优先级和持久化门禁回归测试。
