# Kimix v2.14.16

## 诊断

- 为权限模式切换增加同一 `traceId` 的前端诊断日志，覆盖点击、菜单关闭、延后切换、SDK `setPermission` 前后、UI 权限状态写入前后。
- 为聊天流滚动增加权限切换对齐日志，记录切换事件发生时、下一帧和再下一帧的 `scrollTop` / `scrollHeight` / `clientHeight` / 渲染项数量 / auto-follow 状态。
- 增加 `scrollJumpNearTop` 诊断点，专门捕捉从中下位置突然跳到顶部附近的瞬间。

## 验收建议

- 启动后确认侧栏或设置页版本为 `v2.14.16`。
- 在长会话中反复切换 manual / auto / yolo，复现“瞬间顶到顶部”后导出诊断日志。
- 日志中重点搜索 `[Composer] permissionMode`、`[ChatThread] permissionDiag`、`[ChatThread] permissionDiag rAF1`、`[ChatThread] permissionDiag rAF2`、`[ChatThread] scrollJumpNearTop`。
