# Kimix v2.12.9

- `/custom-theme` 与 `/import-from-cc-codex` 改为优先调用 Kimi Code 0.20.2 官方内置 Skill，不再默认通过长提示词或本地预览模拟官方命令。
- 新增官方 `/mcp-config` Skill 入口；`/theme` 继续打开 Kimix 自己的主题设置，避免调用仅适用于终端的主题选择器。
- 修正 slash 分流顺序：Skill 激活不再重复发送普通消息；压缩、Plan、BTW、撤回、状态和用量命令直接调用官方 Server API；Goal、Swarm、重载在 Server 暂不支持时明确提示，不再伪装成普通消息。
- 官方内置 Skill 不可用时仍保留 Kimix 兼容兜底，并将 custom-theme 兜底更新到当前官方确认流程、路径规则和颜色 token。
