# Kimix v2.8.345 Release Notes

## 长程任务、右侧工具箱与会话恢复修复

本次更新继续打磨 Kimi Code 桌面工作流，重点调整长程任务执行方式、右侧会话工具箱、启动恢复和项目侧栏体验。

### 长程任务

- 长程任务不再展示庞大的下一轮执行提示词预览，减少右侧侧栏噪音。
- 弱化独立审查 agent 流程，默认由执行会话连续推进，最终交给用户做全盘审查。
- 普通对话不再显示长程任务卡片；已归档任务不再进入普通会话侧栏列表。
- 保留用户审查清单，方便任务完成后集中确认风险和残余事项。

### 右侧会话工具箱

- 右侧会话侧栏新增 Git 工具卡，支持查看分支、改动数量、拉取、提交和刷新状态。
- Kimi Code 健康状态迁移到右侧会话侧栏，支持刷新、展开详情、打开登录/模型设置、打开项目目录和重连当前会话。
- 右侧工具卡支持拖动排序，并持久化用户顺序。
- 拖动预览统一为线条落点，不再使用大面积弧形描边。

### 设置与侧栏

- 设置页分区支持拖动排序，并持久化用户顺序。
- 修复主目录为盘根目录时，左侧项目名称为空的问题。
- 移除左侧底部固定的 Kimi Code 健康卡，避免挤占项目列表和设置入口。

### 启动恢复

- 增加本地最近活动项目/会话记录，启动时优先恢复用户上次实际停留的项目和会话。
- 启动恢复会跳过已归档会话，避免打开后落到旧的归档消息里。

### Kimi Code 辅助能力

- 底部状态栏增加 Kimi Code 连接状态文本。
- 进一步补强 Kimi Code 环境、登录、模型、Git 和当前会话绑定状态检测。
