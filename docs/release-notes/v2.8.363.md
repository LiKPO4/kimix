# Kimix v2.8.363

## 修复与改进

- 修复 Kimi Code 正文回填丢失问题，兼容新的 `context.append_loop_event -> content.part` 事件形态。
- 移除长程任务审查 agent 流程，改为执行 agent 连续推进并在结束时提示用户全盘审查。
- 增强右侧工具箱，加入 Git 详情、提交、拉取、推送、远端状态和最近变更视图。
- 修复 Git 状态路径解析丢首字符的问题，避免 `tv_browser` 被显示成 `v_browser`。
- 修复切换项目后 Git 远端状态刷新不及时的问题，区分未配置远端、已配置远端但未跟踪、已设置 upstream。
- 优化 Git 详情打开体验，避免点击详情时短暂闪过默认文本。
- 增加 renderer 卡顿 watchdog 和设置页卡死诊断导出能力。
- 优化右侧工具箱、设置页、侧栏项目/会话同步等交互细节。
