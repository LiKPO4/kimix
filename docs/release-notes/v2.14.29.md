# Kimix v2.14.29

## 改进

- SDK route 会话的 slash 补全开始合并官方 Plugin manifest commands。
- 发送已知的 `/<pluginId>:<commandName>` 时，Kimix 会调用官方 `activatePluginCommand()` 在当前会话内激活，不再需要把这类命令伪装成 Kimix 内置 slash。

## 边界

- Server route 暂不激活 Plugin commands；等官方 Server API 暴露等价能力后再接入，避免把命令发到错误的临时管理会话。
- 未知 slash 仍按原逻辑透传给官方会话，不由 Kimix 抢路由。
