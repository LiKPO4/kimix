# Kimix v2.15.41

## 多 Agent 房间可见正文桥接

- 新 Agent 首次接收任务时默认补入上一轮房间可见正文，能够理解先前 Agent 的最终输出，同时继续使用独立 Provider、模型和官方 session。
- 输入区新增单次正文范围：上一轮、最近 3 轮、选择消息、全部正文和不补充；发送成功后自动恢复上一轮。
- 只共享用户消息和 Agent 最终正文，不共享思考、工具详情、命令结果、审批或用量；同一条正文对同一 Agent 不重复补入。
- 正文范围随 delivery 一起可靠持久化，失败重试保持相同内容；官方历史回放只显示当前用户消息，不暴露隐藏桥接包。
- 单 Agent 补入内容超过 48,000 字时明确要求缩小范围，不静默截断。
