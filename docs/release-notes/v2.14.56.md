# Kimix v2.14.56

## 修复

- 助手消息底部"全部复制"文字大小与状态气泡一致的 13px/18px。
- 历史状态缺少上下文用量时不再伪造显示 `Context: 0.00%`。
- 重启后旧会话切换模型不提示 `Kimi Code session is not active`；恢复原会话、校验项目目录并重试模型切换，同步恢复后的真实 runtime ID。
- 官方事件缺失 `turn.ended` 时已结束回复持续计时：`turn.step.completed` 携带 `finishReason: end_turn` 时收尾 Assistant 显示，工具中间步骤不被误判为完成。
- SDK 因额度不足等错误终止后，子代理、工具和消息状态不再显示运行中。
- AI 正在工作时刷新页面不再错误标记已完成并触发 active turn 冲突；Server snapshot 中仍在生成的正文不再错误标记为 `turn.ended`；正文长时间未更新时后台回读官方 snapshot 并恢复更完整的内容。
- Kimi Web 模式流式思考从多行折叠为单行时，对话视口不再向上跳动。

## 功能

- Swarm 模式锁定时，Composer 底部 Plan 开关左侧显示"Swarm 开"状态按钮。
- 运行中连续 4 秒无正文事件抵达时，主动回读官方历史并在 snapshot 更完整时恢复显示。

## 改进

- Kimi Web 子代理详情左侧竖线移除，子事件类型列固定 104px，避免横向跳动。
- 降低工具运行和命令输出期间聊天滚动热路径的诊断日志与锚点恢复频率，缓解主界面卡顿和计时停滞。
